Pemendek Tautan Pribadi Anda

Dikelola dengan sederhana menggunakan Netlify Redirects.

Bagaimana Cara Kerjanya?

Aplikasi ini tidak menggunakan database. Semua tautan pendek diatur dalam file _redirects.

  1. Untuk menambah tautan baru, edit file _redirects di repositori Anda.
  2. Tambahkan satu baris baru dengan format: /nama-pendek https://url-panjang.com
  3. Commit & Push perubahan ke GitHub.
  4. Netlify akan otomatis mendeploy versi baru, dan tautan Anda langsung aktif!
